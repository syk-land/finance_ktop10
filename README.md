# 야구 인생 (Baseball Career Sim)

주인공이 고교야구부 1학년부터 은퇴까지를 경험하는 웹 기반 커리어 시뮬레이션 게임.
플레이어는 직접 경기를 조작하지 않고 **훈련 방향**과 **시간 흐름**만 통제하며,
실력에 따라 팀·리그가 자동으로 변동되는 구조.

## 실행 방법

순수 HTML/CSS/JS (의존성 없음). 정적 서버에서 바로 실행 가능.

```bash
# 프로젝트 루트에서
python3 -m http.server 8765
# 브라우저에서 http://localhost:8765 접속
```

ES Module을 사용해서 `file://`로 직접 열면 동작하지 않음. 반드시 HTTP 서버 경유.

## 게임 흐름

1. **캐릭터 생성**: 이름·얼굴(6종)·재능(8종)·손잡이(4종) 선택
2. **자동훈련 프리셋 선택**: 슬러거 / 컨택 히터 / 호타준족 / 수비 명인 / 파이어볼러 / 제구파 / 양방향 / 회복 우선
3. **시간 흐름**: ▶ 재생 → 매 500ms마다 1일 진행 (속도 조절 가능 0.5x ~ 4x)
4. **자동 진행**: 평일은 자동 훈련, 주말은 자동 경기 시뮬레이션
5. **일시정지**: 언제든 ⏸ 일시정지로 멈춰서 프리셋 변경·상태 점검
6. **시즌 종료**: 자동 일시정지 → 다음 학년 진입 결정

## 핵심 시스템

| 영역 | 내용 |
|---|---|
| 능력치 | 양방향 (타자 5종 + 투수 5종), cap 150 |
| 재능 타입 | 컨택/파워/주루/수비/만능 / 강속구/제구/변화구 |
| 성장 곡선 | 나이별 훈련 효율 (18세 ×1.6 → 35세 ×0.4) |
| 컨디션 | 호조/슬럼프 사이클, 경기력 보정 |
| 부상 | 경상(1주) / 중상(3주) / 인대(8주) |
| 자동훈련 | 프리셋 가중치 + 부족도 보정 + focusStats 타게팅 |
| 경기 시뮬 | 매치업 공식 (컨택/선구안/구속/제구/변화구 등) |
| 출장 확률 | OVR 곡선 (타자 30~80 / 투수 35~80) + 등판 휴식 |
| 경기 경험치 | 성적별 능력치 보너스 (안타/홈런/삼진/호투 등) |
| 시즌 통계 | 타율/OPS/ERA/WAR 기반 박스스코어 |
| 저장 | localStorage 자동 저장 |

## 파일 구조

```
baseballalone/
├── index.html
├── styles/main.css
├── src/
│   ├── main.js              # 부트스트랩 + tick loop
│   ├── state.js             # 전역 상태 + 세이브
│   ├── data/
│   │   ├── names.js         # 한국 성/이름 풀
│   │   └── teams.js         # 리그/팀 템플릿
│   ├── systems/
│   │   ├── player.js        # 능력치·훈련·성장·부상
│   │   ├── npc.js           # NPC 풀 생성
│   │   ├── league.js        # 리그/일정/순위
│   │   ├── simulator.js     # 경기 자동 시뮬레이션
│   │   ├── week.js          # 주 단위 진행
│   │   ├── career.js        # 학년/단계 분기
│   │   ├── autoTrain.js     # 자동훈련 프리셋
│   │   └── tick.js          # 실시간 1일 진행
│   ├── views/
│   │   ├── menu.js          # 캐릭터 생성 + 메뉴
│   │   └── weekly.js        # 메인 게임 화면
│   └── render/
│       ├── svg.js           # SVG 헬퍼
│       ├── avatars.js       # 얼굴 6종
│       ├── character.js     # 3등신 캐릭터 (자세별)
│       └── radar.js         # 레이더 차트
└── assets/                  # 추후 이미지 교체 슬롯
```

## 로드맵

- **Phase 1 (현재)**: 코어 루프 — 캐릭터, 능력치, 훈련, 경기 시뮬, 시즌, 세이브
- **Phase 2**: POV 시각화 — 타석/투구 1인칭, 야구장 탑뷰, 공 궤적
- **Phase 3**: 커리어 깊이 — 드래프트, 1·2군 콜업, NPC 성장/은퇴, 수상, 통산, 명예의 전당, 해외 진출
- **Phase 4**: 폴리시 — 일기/뉴스, 도전과제, 날씨, 사운드
