# 🌄 PROJECT 2100 - 지역별 전투 배경 이미지 생성 프롬프트 목록
(14 Regional Post-Apocalyptic Combat Background Prompts)

이 문서는 AI 이미지 생성 툴(`generate_image`)을 사용하여 각 지역(1~14)의 전투 배경 이미지를 제작할 때 사용하는 영문 프롬프트와 디자인 기획을 정리한 가이드라인입니다.

## 🎨 공통 디자인 원칙 (Shared Design Principles)
- **전체 톤앤매너:** 아포칼립스 이후의 세상을 다루며, 눈 또는 낙진(Fallout Ash)이 덮여 전체적으로 **밝고 하얀색(Bright & White)**이 지배적인 분위기를 만듭니다.
- **아트워크 스타일:** 2D 횡스크롤 게임 배경(`flat 2D side-scroller game background`), 간결하면서 대비가 강한 미니멀 일러스트 스타일(`minimalist high-contrast art style, clean illustration`), 그리고 실루엣과 외곽선 위주의 모던한 그래픽을 추구합니다.
